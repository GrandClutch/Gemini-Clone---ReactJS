# Gemini Clone - ReactJS 
## Objective
1. Render prompt and input using Gemni's API
2. Practice React Route with the Settings and New Chat in the Sidebar
## How It Works
1. Give a prompt to Gemini in the input field
2. Can use the suggested prompts on home screen
3. Storing recent prompts to Recent in the Sidebar
4. New Chat in the Sidebar for new prompt or input
